# 物体检测逐步收缩阈值优化方案

## 一、核心思想

通过**逐步收紧阈值**的方式，自适应找到稳定的阈值：

```
50mm → 不稳定则 +5mm → 55mm → 不稳定则 +5mm → 60mm → ... → 稳定后锁定
```

一旦找到稳定阈值，后续不再变化。

---

## 二、阈值调整机制

### 初始状态
```
当前阈值 = 50mm
不稳定计数 = 0
稳定计数 = 0
阈值已锁定 = false
```

### 每帧检查逻辑

```
┌─────────────────────────────────────────────────────────────┐
│  计算边界变化                                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  xSpan变化 = |xSpan - lastXSpan|                            │
│  ySpan变化 = |ySpan - lastYSpan|                            │
│  像素变化 = max(xSpan变化, ySpan变化)                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  判断稳定性                                                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  if (像素变化 >= 3):     // 变化超过2个，即>=3                │
│      不稳定计数++                                            │
│      稳定计数 = 0        // 重置稳定计数                       │
│                                                             │
│      if (不稳定计数 >= 3 AND 阈值未锁定):                     │
│          当前阈值 += 5mm      // 收紧阈值                     │
│          不稳定计数 = 0      // 重置，重新观察                 │
│          Log("阈值收紧至 " + 当前阈值 + "mm")                 │
│                                                             │
│  else:  // 像素变化 <= 2                                     │
│      稳定计数++                                              │
│      // 不稳定计数不重置，是累计的                             │
│                                                             │
│      if (稳定计数 >= 10 AND 阈值未锁定):                      │
│          阈值已锁定 = true                                   │
│          Log("阈值锁定在 " + 当前阈值 + "mm")                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  更新上一帧数据                                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  lastXSpan = xSpan                                          │
│  lastYSpan = ySpan                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

```

                    │  阈值 = 50mm        │

                               │
            ┌──────────────────┼──────────────────┐
            ↓                  ↓                  ↓

            ↓                  │                  ↓
     阈值 += 5mm               │           阈值已锁定 = true

                      ↓                           │

                      ↓                           │

                      ↓                           │

```

## 五、关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 初始阈值 | 50mm | 起始阈值 |
| 阈值增量 | 5mm | 每次收紧增加 |
| 触发收紧 | 累计 3 次 | 像素变化 >= 3（不是连续） |
| 触发锁定 | 累计 10 次 | 像素变化 <= 2（不是连续） |
| 像素变化判定 | >= 3 | xSpan 或 ySpan 变化 >= 3 为不稳定 |
| 阈值上限 | 可选 | 可设置最大阈值，如 100mm |

## 六、与之前方案对比

| 特性 | 之前方案（两状态） | 新方案（逐步收缩） |
|------|------------------|------------------|
| 阈值数量 | 2个（50mm / maxDiff×0.6） | 多个（50mm 起逐步+5mm） |
| 状态数 | 2个（稳定/不稳定） | 1个（逐步收敛到稳定） |
| 恢复机制 | 需要连续15次稳定 | 累计10次稳定即锁定 |
| 自适应性 | 较弱 | 更强，自适应找到最佳阈值 |
| 复杂度 | 中等 | 简单 |

## 七、实现要点

### 新增成员变量
```java
// 阈值控制
private float currentMaskThreshold = 50.0f;  // 当前阈值，初始50mm
private int unstableCount = 0;               // 不稳定计数（累计）
private int stableCount = 0;                 // 稳定计数（累计）
private boolean thresholdLocked = false;     // 阈值是否已锁定

// 上一帧边界
private int lastXSpan = 0;
private int lastYSpan = 0;

// 可选：阈值上限
private static final float MAX_THRESHOLD = 100.0f;
```

### 核心逻辑
```java
void updateThreshold(int xSpan, int ySpan) {
    if (thresholdLocked) {
        return;  // 已锁定，不再调整
    }

    int xChange = Math.abs(xSpan - lastXSpan);
    int yChange = Math.abs(ySpan - lastYSpan);
    int pixelChange = Math.max(xChange, yChange);

    if (pixelChange >= 3) {
        // 不稳定
        unstableCount++;
        stableCount = 0;  // 重置稳定计数

        if (unstableCount >= 3) {
            currentMaskThreshold += 5.0f;
            if (currentMaskThreshold > MAX_THRESHOLD) {
                currentMaskThreshold = MAX_THRESHOLD;
            }
            // 阈值调整后，重置所有计数，重新观察
            unstableCount = 0;
            stableCount = 0;
            Log.d(TAG, "阈值收紧至: " + currentMaskThreshold + "mm");
        }
    } else {
        // 稳定 - 不稳定计数不重置（累计的）
        stableCount++;

        if (stableCount >= 10) {
            thresholdLocked = true;
            Log.d(TAG, "阈值锁定在: " + currentMaskThreshold + "mm");
        }
    }

    lastXSpan = xSpan;
    lastYSpan = ySpan;
}
```

## 八、预期效果

```
场景1：矮物体（边界本身稳定）
  50mm → 连续稳定10次 → 锁定在 50mm

场景2：中等物体（轻微跳动）
  50mm → 不稳定3次 → 55mm → 稳定10次 → 锁定在 55mm

场景3：高物体（明显跳动）
  50mm → 不稳定3次 → 55mm → 不稳定3次 → 60mm → 不稳定3次 → 65mm → 稳定10次 → 锁定在 65mm

场景4：极端情况
  阈值会持续上升直到达到上限（如100mm）或稳定为止
```

---

## 九、优势

1. **自适应**：自动找到最适合当前物体的阈值
2. **简单**：单一状态，逻辑清晰
3. **渐进式**：阈值逐步收紧，不会突变
4. **可靠性**：锁定后不再变化，确保稳定
