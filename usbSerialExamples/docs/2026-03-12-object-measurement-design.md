# 物体尺寸测量 Activity 设计文档

> 创建日期: 2026-03-12
> 状态: 设计完成，待实现

---

## 一、功能概述

创建一个新的 Activity，用于测量放置在桌面上的物体的三维尺寸（长、宽、高）。

**核心流程：**
1. 校准阶段：扫描空桌面，获取稳定的基线深度数据
2. 测量阶段：放置物体后，计算物体尺寸，等待稳定后输出最终结果

---

## 二、设计决策

| 项目 | 决定 | 说明 |
|-----|------|------|
| 稳定性判断 | **体积差异** | `\|V_curr - V_prev\| < 阈值` |
| Baseline存储 | **文件存储** | JSON格式，保存到内部存储 |
| 采样方式 | **实时帧取最新** | 利用现有15fps数据流，每0.5秒取最新帧 |
| 采样间隔 | **0.5秒** | 平衡响应速度和稳定性 |
| 稳定持续时间 | **2秒** | 连续4次采样都稳定才算稳定 |

### 4.2 稳定性判断逻辑

```java
// 伪代码
float avgCurrent = calculateAverage(currentDepth);
float avgDiff = Math.abs(avgCurrent - avgPrevious);

if (avgDiff < BASELINE_STABLE_THRESHOLD) {
    stableCount++;
    if (stableCount >= 4) {  // 4 * 0.5s = 2s
        // 稳定，保存Baseline
        saveBaseline(currentDepth);
        enableNextButton();
    }
} else {
    stableCount = 0;
}
avgPrevious = avgCurrent;
```

## 十一、注意事项

1. **像素尺寸 (pixelSizeX, pixelSizeY)** 需要根据实际传感器参数和安装距离测量确定
2. **阈值参数** 需要通过实际测试调整
3. **45° 视角** 的透视畸变在粗略估算中暂时忽略
4. **UNIT=0 模式** 的距离公式：`距离(mm) = (像素值 / 5.1)²`
